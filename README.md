**Architect**: Stiff McGriff

**License**: Open Source (Free for all use)

## Overview
Omni-Studio is a next-generation, AI-powered Digital Audio Workstation (DAW) built entirely with free, open-source tools. It is designed to replicate and exceed the functionality of paid industry standards like DaVinci Resolve, LANDR, and Output's Co-Producer without costing a dime.

## Feature Set (Roadmap)
- [x] **Core Engine**: Python + PyQt6 GUI
- [x] **AI Mastering**: Matchering 2.0 integration (LANDR mimic)
- [x] **Stem Separation**: Spleeter/Demucs (Polymath mimic)
- [x] **Audio Analysis**: Librosa-based BPM/Key detection
- [ ] **Generative AI**: Riffusion/ACE-Step (Suno mimic)
- [ ] **VST Host**: Automatic plugin scanning and loading
- [ ] **Co-Producer**: AI sample suggestion based on harmonic analysis

## Installation
1. Clone/Download this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python main.py`
